Panes

Close a terminal pane with C-wC-c

To change two vertically split windows to horizonally split C-wtC-wK

Horizontally to vertically: C-wtC-wH

C-wt makes the first (topleft) window current C-wH moves the current window to full-height at far left C-wK moves the current window to full-width at the very top

Comment line (via plugin vim-commentary): gcc in normal mode, gc in visual mode
