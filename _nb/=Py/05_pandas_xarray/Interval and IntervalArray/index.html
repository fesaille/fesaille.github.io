    Interval¶

    Pandas IntervalDtype extends Numpy ndarray type. This is not an actual numpy dtype, but a duck type.

    An Interval is an immutable object implementing a bounded slice-like interval.

    Parameters: - left: scalar - right: scalar - closed: string, default to right
