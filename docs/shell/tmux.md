# tmux


## Panes

### Layout

**Change layout**: `select-layout` has five predefined layouts : `even-horizontal`, `even-vertical`, `main-horizontal`, `main-vertical`, or `tiled`.

**Move panes** `move-pane [-t dst-pane]`
