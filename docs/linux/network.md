# Network

## Basics

The [OSI (Open Systems Interconnection)](https://en.wikipedia.org/wiki/OSI_model) uses seven layers, 
- physical layer (layer 1)
- network layer (layer 3)
- transport layer (layer 4)
- application layer (layer 7)
